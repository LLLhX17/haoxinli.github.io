Autonomous Needle Insertion for Common Femoral Artery Access Robotic System

During my time at the Advanced Micro and Nanosystems Laboratory at the University of Toronto, under Prof. Sun Yu, I contributed to designing a needle insertion robot aimed at enabling minimally invasive surgeries for cardiovascular patients in rural areas. The robot, equipped with a 3 DOF coarse stage for general movements and a 4 DOF fine stage for precision adjustments, was equipped with an IR sensor and a US probe for detailed artery examination. My role in this project was primarily researching and procuring components for the robot prototype. By the end of my tenure, we had completed CAD work and design for manufacturing, paving the way for future prototyping and assembly.

NLP, Python, Data Scientist, UofT

Sentiment Analysis: Russo-Ukrainian War

In this project, we used Natural Language Processing (NLP) and sentiment analysis to understand public sentiment regarding the 2022 Russo-Ukrainian conflict, as reflected in over 550,000 social media posts. Key tasks included preprocessing the large dataset to remove noise, normalize text, and ensure data quality, thereby improving model performance. We conducted sentiment analysis using several machine learning models, with an Artificial Neural Network (ANN) achieving the highest accuracy of 93.4%. Further, we analyzed word occurrences and their correlations using Mutual Information (MI) and Pointwise Mutual Information (PMI) to deepen our understanding of the public sentiment. With additional tools such as Word Cloud and Latent Dirichlet Allocation (LDA) combined with ChatGPT, we revealed prevalent themes and hidden topics in the conversations. The insights drawn will be valuable to the Ukrainian government and relevant NGOs in strategizing their actions for improving Ukraine's international presence.

Curriculum Design using Natural Language Processing

This project involved designing a Master of Data Science curriculum based on job market demands. I gathered and cleaned job data from Indeed.com, extracted skills using OpenAI's GPT-3.5-turbo model, and determined their frequency using N-grams. I computed skill similarity using Levenshtein distance and grouped similar job postings using KMeans clustering. The result was a data science-focused curriculum tailored to industry needs and trends.

Deep Learning in Face Mask Image Classification

This project is aimed at automating facial mask detection using machine learning techniques to enhance public health measures amid the COVID-19 pandemic. Utilizing a Convolutional Neural Network (CNN) model, the team processed and trained a dataset of approximately 11.8k images from Kaggle to identify individuals wearing masks. The model, composed of 7 layers, achieved a high testing accuracy of 99.8% on the original dataset and maintained high performance (99.54% accuracy) on a separate test dataset.

Mechatronic System: Design and Integration

A project that aimed at designing a mechatronic system for a Turtlebot to navigate unknown environments. Leveraging the power of Robot Operating System (ROS), the team successfully simulated the robot's autonomous navigation. To enhance the bot's usefulness, the team also developed an innovative feature for the robot to identify and interact with emotional victims. This was achieved by implementing advanced image identification algorithms and utilizing OpenCV for image detection. The extensive programming in C++ incorporated elements like odometry, gyroscope, bumper, and an RGB camera for the Turtlebot's exploration, which resulted in the robot being capable of exploring 85% of unknown environments autonomously.
